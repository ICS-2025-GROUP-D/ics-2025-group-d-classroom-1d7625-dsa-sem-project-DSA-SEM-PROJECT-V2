## Stack Implementation Complexity

| Operation | Time Complexity | Space Complexity |
|-----------|------------------|------------------|
| push()    | O(1)             | O(1)             |
| pop()     | O(1)             | O(1)             |
| peek()    | O(1)             | O(1)             |
| is_empty()| O(1)             | O(1)             |
